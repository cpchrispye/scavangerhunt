How to Use This Tool

Encrypting Your JSON

  1. Select your questions.json file or paste its content in the text area
  2. Enter a strong password
  3. Click "Encrypt"
  4. Download the encrypted file (recommended filename: questions.enc.json)

Using With Your Quiz

Update your quiz URL to include the password parameter:

https://yourdomain.com/index.html?q=q1&p=yourpassword

About the Encryption

This tool uses AES-256-GCM encryption, which is a secure authenticated encryption algorithm. The password is used to derive a key using PBKDF2 with 100,000 iterations.

File Format

The encrypted file uses a simple custom format with a JSON structure containing:

  • Salt: Used for key derivation
  • IV: Initialization vector for AES
  • Data: The encrypted JSON data
  • Auth tag: To verify the integrity of the data
